# Jyoshna Naluvala — Data Analyst Portfolio

This is my personal Data Analyst portfolio website, designed to showcase my experience, technical skills, projects, and career journey in a clear and interactive way.

🔗 **Live Portfolio:** https://jnaluvala.github.io/portfolio/

---

## About This Portfolio

This portfolio was built to present my work as a Data Analyst in a more practical and stakeholder-focused way. Instead of only listing tools and technologies, the website highlights how I approach data problems, validate information, explain insights, and support better business decisions.

The main theme of the portfolio is:

> I help stakeholders move from confusion to clarity by showing what changed, why it matters, and what to do next.
